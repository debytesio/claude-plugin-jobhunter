# Contributing to Job Hunter

Thanks for your interest in contributing! Here's how to get involved.

## Reporting Bugs

Open a [bug report](https://github.com/debytesio/claude-plugin-jobhunter/issues/new?template=bug_report.md) with:
- Steps to reproduce
- Expected vs actual behaviour
- Your environment (OS, Python version, plugin version)

## Requesting Features

Open a [feature request](https://github.com/debytesio/claude-plugin-jobhunter/issues/new?template=feature_request.md) describing your use case and proposed solution.

## Questions & Discussion

Join us on [Discord](https://discord.gg/8BZZ2fQGhu) for questions, feedback, and general discussion.

## Pull Requests

1. Open an issue first to discuss the change
2. Fork the repo and create a branch from `main`
3. Make your changes and test locally
4. Submit a PR linking to the issue

## Code of Conduct

Be respectful, constructive, and inclusive. We're building something useful together.
